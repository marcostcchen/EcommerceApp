## Installation

1. Clone the repository:

  ```bash
  git clone https://github.com/your-username/ecommerce-platform.git
  ```

2. Change to the project directory:

  ```bash
  cd ecommerce-platform
  ```

3. Install dependencies:

  ```bash
  flutter pub get
  ```

4. Run the app:

  ```bash
  flutter run
  ```
